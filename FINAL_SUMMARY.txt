## 💻 HOW TO EDIT THINGS

### Change Your Name
Edit: `components/sections/About.js`
```javascript
<h3>Siddharth Kumar</h3>  // Change this
```

### Change Colors
Edit: `styles/globals.css`
```css
--accent: #818cf8;  // Change this to your color
```

### Change Contact Info
Edit: `components/sections/Contact.js`
```javascript
<span>your@email.com</span>  // Change this
```

### Add New Section
1. Create `components/sections/NewSection.js`
2. Create `components/sections/NewSection.module.css`
3. Import in `app/page.js`
